# Digital Services

Technologies you can use when building and running government services

  • Rehnuma is an API gateway (opens new window) that sits between the citizens and backend services provided by each department. It acts as the primary aggregator and transformer for G2G and G2C services and provides a number of functions ranging from authentication to caching, throttling, and monitoring.

  • Pehchaan is the digital identity (opens new window) of the citizens of Khyber Pakhtunkhwa. It is an OpenID Connect (opens new window) implementation for use as centralized authentication and authorization mechanism for accessing government services. Citizens sign up for a digital identity ONCE, and use the same identity across all government services.

  • Paighaam allows government departments to communicate with the citizens through a unified omni-channel communication gateway (opens new window). This system greatly simplifes messaging and provides a communication platform for sending emails, text messages, and push notifications with ease.
